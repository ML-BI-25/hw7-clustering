## Проверка задания (автотесты)

Проверка задания осуществляется автоматически, а потом при личной встрече.

Автотест:

1. Создаёт чистое Python-окружение;
2. Устанавливает все зависимости из файла `requirements.txt`;
3. Запускает ноутбук `notebooks/homework7.ipynb` целиком сверху вниз.

### Требования

- ноутбук должен выполняться без ошибок;
- все используемые библиотеки должны быть указаны в `requirements.txt`;
- код не должен требовать ручного ввода или интерактивных действий;
- пути к файлам должны быть относительными и указывать на файлы внутри репозитория.

Если выполнение завершается с ошибкой, автотест считается не пройденным.

### Tips

- перед сдачей перезапустите kernel и выполните ноутбук целиком  
  (*Restart Kernel & Run All*);
- не используйте локальные пути вида `C:\...` или `/Users/...`;
- если код работает локально, но не проходит автотест — проверьте, что все используемые пакеты указаны в `requirements.txt`.
